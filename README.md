# Visualization of the Least Recently Used algorithm using OpenGL

Just a simple animation of the LRU algorithm

## Instructions to run in Ubuntu:

1) g++ lru.cpp -o gl -lGL -lGLU -lglut
2) ./gl

Press 'q' to quit
